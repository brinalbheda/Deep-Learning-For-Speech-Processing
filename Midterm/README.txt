Midterm folder contains 1 file - midterm.py

--Steps to run the code:

1. Change the directory to where the input files will be saved in the script on line 25.
2. Run midterm.py from terminal.

--What does this code do?

1. Detecting phonemes for every frame.
2. Extracting audio sample features by converting time to frequency domain and extracting MFCCs from frequency domain signals.
3. Defining and training the neural network.
4. Testing the neural network model with different samples.
5. Plotting the convergence graph for epochs vs loss.
